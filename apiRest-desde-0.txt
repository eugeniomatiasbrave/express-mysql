
PASOS PARA GREAR API REST DESDE CERO:

1- Instalar package.json: npm init -y  
2- Instalar express: npm i express
3- Crear el "type":"module"  en package.json
4- Crear carpeta SRC.
5- Crear archivo raiz..llamarlo index.js, app.js o server.js.
6- Realizar la importacion de express en archivo raiz.
7- Instalar nodemon para levantar el servidor, o con watch de node:  npm i nodemon -D en devDependencies
8- conectarme a una base de datos: Mongodb , mySQL en app.js 
9- Crear script "dev" para levantar el servidor con nodemon: "dev": "nodemon src/app.js
10- Crear el servidor en app.js y levantarlo en algun puerto.
11- crear el .gitignore si deseo subirlo a github: dento minimo ( node_module , .env) 
12- crear las variables de entorno su creo necesario en .env para conectarme a la db.
13- crear carpeta router.
14- crear las rutas en un archivo (endpoints).
15- Modelo  controller, servicio
16- crear carpeta controllers
